\doxysubsection{Detailed Description}
Classe responsável pelo gerenciamento do jogo Reverse, derivada da classe \mbox{\hyperlink{classJogo}{Jogo}}. 

A classe \mbox{\hyperlink{classReversi}{Reversi}} gerencia o estado e as regras do jogo \mbox{\hyperlink{classReversi}{Reversi}}. Ela herda todos os métodos da classe \mbox{\hyperlink{classJogo}{Jogo}} e implementa funcionalidades específicas para o jogo \mbox{\hyperlink{classReversi}{Reversi}}, como a alternância de jogadores, validação de jogadas, realização de jogadas e verificação de vitória.

Este método verifica se a jogada na posição especificada (linha e coluna) é válida para o jogador atual. Se a jogada for válida, o método atualiza o tabuleiro, vira as peças do oponente que foram capturadas, alterna para o próximo jogador e imprime o estado atualizado do tabuleiro.

O método funciona da seguinte maneira\+:
\begin{DoxyItemize}
\item Verifica se a jogada é válida chamando o método \textquotesingle{}jogada\+Valida\textquotesingle{} para a posição especificada.
\item Se a jogada for válida\+:
\begin{DoxyItemize}
\item A peça do jogador atual é colocada na posição indicada no tabuleiro.
\item O método \textquotesingle{}virar\+Pecas\textquotesingle{} é chamado para capturar as peças do oponente.
\item O jogador atual é alternado utilizando o método \textquotesingle{}alternar\+Jogador\textquotesingle{}.
\item O tabuleiro atualizado é impresso chamando o método \textquotesingle{}imprimir\+Tabuleiro\textquotesingle{}.
\end{DoxyItemize}
\item Se a jogada não for válida, uma mensagem de erro é exibida no console.
\end{DoxyItemize}

Verifica se o jogo terminou por falta de jogadas válidas. 

\begin{DoxyReturn}{Returns}
\textquotesingle{}true\textquotesingle{} se houve vitória,\textquotesingle{}false\textquotesingle{} caso contrário.
\end{DoxyReturn}
Este método percorre todo o tabuleiro e verifica se ainda há jogadas válidas disponíveis para qualquer um dos jogadores (\textquotesingle{}X\textquotesingle{} ou \textquotesingle{}O\textquotesingle{}). O jogo é considerado terminado se não houver jogadas válidas restantes para nenhum dos jogadores.

O método funciona da seguinte forma\+:
\begin{DoxyItemize}
\item Para cada posição do tabuleiro 8x8, verifica se a posição permite uma jogada válida para \textquotesingle{}X\textquotesingle{} ou \textquotesingle{}O\textquotesingle{}.
\item Se for encontrada pelo menos uma jogada válida, o método retorna {\ttfamily false}, indicando que o jogo pode continuar.
\item Se nenhuma jogada válida for encontrada em todo o tabuleiro, o método retorna {\ttfamily true}, indicando que o jogo terminou.
\end{DoxyItemize}

\begin{DoxyReturn}{Returns}
{\ttfamily true} se o jogo terminou (nenhuma jogada válida restante para qualquer jogador), {\ttfamily false} caso contrário. 
\end{DoxyReturn}
