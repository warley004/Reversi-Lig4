\doxysubsection{Detailed Description}
Classe responsável pelo gerenciamento do jogo \mbox{\hyperlink{classLig4}{Lig4}}, derivada da classe \mbox{\hyperlink{classJogo}{Jogo}}. 

A classe \mbox{\hyperlink{classLig4}{Lig4}} gerencia o estado e as regras do jogo \mbox{\hyperlink{classLig4}{Lig4}}. Ela herda todos os métodos da classe \mbox{\hyperlink{classJogo}{Jogo}} e implementa funcionalidades específicas para o jogo \mbox{\hyperlink{classLig4}{Lig4}}, como a alternância de jogadores, validação de jogadas, realização de jogadas e verificação de vitória.

Este método percorre todo o tabuleiro de \mbox{\hyperlink{classLig4}{Lig4}}. Para cada célula que não está vazia, ele verifica se há uma sequência de quatro peças consecutivas em qualquer uma das direções possíveis (horizontal, vertical, diagonal para cima, ou diagonal para baixo). Se for encontrada uma sequência vencedora, a função retorna \textquotesingle{}true\textquotesingle{}.
